# AI 候補者評価アプリ Ver.1.2 🎉

## 👋 やあ！みんな！このアプリって、どんなん？

このアプリはね、会社がお仕事してくれる仲間を探すときに、その人が会社に合うかどうか、AIっていうかしこいロボットが、点数をつけたり、いいところやちょっと心配なところを教えてくれるのを助ける道具なんだ！

履歴書とか自己PRっていう、長いお手紙みたいなのを、このアプリにペタッと貼ると、AIがそれを分かりやすく整理して、パッと見て「この人はこんな人かな？」って分かるようにしてくれるんだ。すごいよね！

---

## ✨ 自分のパソコンでアプリを動かす準備をしよう！

このアプリのプログラムをもらっただけじゃ、まだ動かないんだ。動かすためには、ちょっと準備が必要だよ。パソコンで特別な黒い画面（ターミナルって言うんだ）を使って、順番に準備していくよ！

**最初に確認すること！**

このアプリを動かすために、みんなのパソコンに、次の３つの道具が入ってるか見てみてね。もし入ってなかったら、おうちの人に手伝ってもらって、先に入れてきてね！

1.  **Python (パイソン):** 🐍
    *   **これ、なーに？:** プログラムのおしゃべり言葉みたいなものだよ。このアプリの裏側で動いてるんだ。3.9より大きい数字のバージョンが必要だよ。
    *   **入ってるかチェック！:** 黒い画面で `python --version` って打ってみて。もしエラーが出たら、入ってないってことだよ。
    *   **入れ方:**
        *   Pythonの公式サイトからインストーラーをダウンロードするのがおすすめだよ。[https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)
        *   ダウンロードしたファイルを開いて、画面の言うとおりに進んでね。**大事なこと！** 最初の画面で **「Add Python X.X to PATH」** っていうチェックボックスがあったら、**絶対にチェックを入れてね！** これを忘れると、後で動かなくなっちゃうことがあるんだ。
        *   Microsoft Store からも入れられるけど、公式サイトから入れる方が確実だよ。

2.  **Node.js (ノードジェイエス) と npm (エヌピーエム):** 💻
    *   **これ、なーに？:** アプリの見た目を作るための、道具箱みたいなものだよ。Node.js を入れると、npmっていう部品管理ツールも一緒についてくるんだ。Node.js の 18 より大きい数字のバージョンが必要だよ。
    *   **入ってるかチェック！:** 黒い画面で `node -v` と `npm -v` をそれぞれ打ってみて。もしエラーが出たら、入ってないってことだよ。
    *   **入れ方:**
        *   Node.js公式サイトからインストーラーをダウンロードするのが一番だよ。[https://nodejs.org/ja](https://nodejs.org/ja)
        *   **「LTS」** って書いてある方（安定してるバージョンだよ）のインストーラーをダウンロードして開いてね。
        *   画面の言うとおりに進んでいけば、インストールできるよ。

3.  **Git (ギット):** 🐙
    *   **これ、なーに？:** プログラムの変更を記録したり、他の場所からプログラムをコピーしてきたりするための道具だよ。タイムマシンみたいなものかな。
    *   **入ってるかチェック！:** 黒い画面で `git --version` って打ってみて。もしエラーが出たら、入ってないってことだよ。
    *   **入れ方:**
        *   Git公式サイトからインストーラーをダウンロードしてね。[https://git-scm.com/download/win](https://git-scm.com/download/win)
        *   ダウンロードしたファイルを開いて、画面の言うとおりに進んでね。設定がいっぱい出てくるけど、分からなかったら、**そのまま「Next」を押し続けて大丈夫**だよ。

**↑↑↑ ３つの道具がちゃんと入ったら、いよいよ準備開始だよ！ ↑↑↑**

**準備のステップ:**

1.  **プログラムをゲット！ (Git Clone)**
    *   まず、このアプリのプログラムをみんなのパソコンにコピーするよ。黒い画面で、下のコマンドを打つんだ。（もうやってるかもしれないね！）
        ```bash
        git clone https://github.com/pattarawady/yadoriki_recruit.git
        cd yadoriki_recruit
        ```
    *   これで `yadoriki_recruit` っていう名前のフォルダの中に、プログラムが入ったよ。

2.  **バックエンドの準備 (台所の準備だよ！)**
    *   まず、バックエンド（裏側）の部品が入ってるフォルダに移動するよ。
        ```bash
        cd backend
        ```
    *   次に、このアプリ専用の道具箱（仮想環境って言うんだ）を作るよ。
        ```bash
        python -m venv venv
        ```
    *   作った道具箱を使えるようにするよ！（WindowsのPowerShellならこれ↓）
        ```powershell
        .\venv\Scripts\Activate.ps1
        ```
        (コマンドプロンプトなら `.\venv\Scripts\activate`、Mac/Linuxなら `source venv/bin/activate`)
        黒い画面の行の最初に `(venv)` って出たら成功だよ！
    *   必要な部品をインターネットから持ってくるよ。
        ```bash
        pip install -r requirements.txt
        ```
    *   **大事なカギの設定！** `.env.example` をコピーして `.env` っていう名前に変えるんだ。
        ```powershell
        Copy-Item .env.example .env
        ```
        そして、新しくできた `.env` ファイルをメモ帳で開いて、`GEMINI_API_KEY=` のところに、**Googleからもらった秘密のAPIキー**を貼り付けて保存するんだ。**このファイルは、絶対に人に教えちゃダメだよ！**
    *   これでバックエンドの準備はOK！ 一旦、元の場所に戻ろう。
        ```bash
        cd ..
        ```

3.  **フロントエンドの準備 (お店の見た目の準備だよ！)**
    *   今度は、フロントエンド（みんなが見えるところ）の部品が入ってるフォルダに移動するよ。
        ```bash
        cd frontend
        ```
    *   こっちも必要な部品をインターネットから持ってくるよ。
        ```bash
        npm install
        ```
        (WARNがいっぱい出ても気にしなくて大丈夫)
    *   こっちにもカギの設定が必要だよ。`.env` っていうファイルを作って（もし無かったら）、下の1行を書くんだ。
        ```
        REACT_APP_API_URL=http://localhost:5001/api
        ```
        ファイルを保存してね。
    *   Tailwind CSS の設定ファイルを作るよ（もうあるかもしれないけど）。
        ```bash
        npx tailwindcss init -p
        ```
        (「already exists」って出たらOK)
    *   `tailwind.config.js` を開いて、`content` をこう書き換えるんだ。
        ```javascript
        content: ["./src/**/*.{js,jsx,ts,tsx}"],
        ```
    *   `src/index.css` を開いて、中身を全部消してから、下の3行を書くんだ。
        ```css
        @tailwind base;
        @tailwind components;
        @tailwind utilities;
        ```
    *   これでフロントエンドの準備もOK！

4.  **アプリを起動！**
    *   これでやっとアプリが動かせるよ！ 黒い画面を2つ開いてね。
    *   **1つ目の黒い画面 (バックエンド用):**
        ```bash
        cd backend
        .\venv\Scripts\Activate.ps1
        python run.py
        ```
    *   **2つ目の黒い画面 (フロントエンド用):**
        ```bash
        cd frontend
        npm start
        ```
    *   ブラウザで `http://localhost:3000` を開いてみてね。

これで、みんなのパソコンでこのアプリが動くはずだよ！

## 🚀 アプリの使い方

使い方はとっても簡単だよ！

1.  **情報を貼り付け！**
    *   画面の左上にある大きな四角（テキストエリアって言うんだ）に、調べたい人の情報をコピーして、ペタッと貼り付けるんだ。（Ctrl+Vで貼り付けられるよ！）
    *   関係ない情報も、AIが賢く消してくれるから、気にしないでいいよ。

2.  **「解析する」ボタン、ポチッ！**
    *   情報を貼り付けたら、その下にある「▶︎ 解析する」って書いてるボタンを押すんだ。
    *   ボタンが「解析中...」に変わって、ちょっと待っててね。AIが頑張って考えてる時間だよ。

3.  **結果を見る！**
    *   しばらくすると、画面の右側に結果が表示されるよ！
        *   **マッチ度:** その人が会社にどれくらい合ってるかの点数（％）が大きく表示されるよ。点が高いほど「この人は会社に合いそう！」ってことだよ。
        *   **評価詳細:** 「必須条件」「優遇条件」「その他」みたいに分かれてて、それぞれの項目でAIがどう思ったかと、理由が書いてある表が出てくるよ。色もついてて分かりやすいでしょ？
        *   **総評:** 一番下に、AIが全体を見てどう思ったかのコメントが書いてあるよ。これも参考にしてね。

4.  **履歴を見る！**
    *   解析するたびに、結果は画面左下の「履歴」にどんどん溜まっていくよ。
    *   人の名前と点数がリストになってるから、前に解析した人の結果をもう一度見たかったら、その人の名前をクリックすると、右側の詳しい表示が変わるよ！

5.  **結果を保存！ (エクスポート)**
    *   履歴に溜まった結果を、自分のパソコンにファイルとして保存できるんだ。
    *   左側の「結果をエクスポート」ってところで、「CSV」か「Markdown」かを選ぶんだ。（CSVはExcelとかで開けるやつ、Markdownはメモ帳とかで見やすいやつだよ）
    *   形式を選んだら、「▶︎ X件 結果エクスポート」ボタンを押すんだ。そしたらファイルがダウンロードされるよ！
    *   **大事な注意！:** CSVファイルをExcelで開いたとき、もし文字がグチャグチャになったら、Excelの「データ」タブ → 「テキストまたはCSVから」を選んで、文字コードで「UTF-8」ってやつを選んで開いてみてね！

## 🔧 ルールを変えたいとき (ちょっと難しいかも)

「この項目はもっと大事！」「新しい評価項目を追加したい！」って思うこともあるかもしれないね。そういう時は、プログラムの中にある「ルールブック」を書き換える必要があるんだ。

*   **どこにあるの？:** プログラムの中の `backend/app/config.py` っていうファイルが、そのルールブックだよ。
*   **何が書いてるの？:**
    *   `CRITERIA_PROMPT`: AIに「こういう基準で見てね」ってお願いしてる文章が全部書いてあるんだ。ここの文章を変えると、AIの評価の仕方が変わるよ。
    *   `SCORE_MAP`: ◎が何点、〇が何点、みたいな点数のルールが書いてあるんだ。
    *   `CATEGORY_WEIGHTS`: 「必須条件は何％分」「優遇条件は何％分」みたいな、カテゴリごとの大事さが書いてあるんだ。
    *   `ITEM_WEIGHTS`: 各カテゴリの中の、さらに細かい項目（年齢とか経験とか）のそれぞれの重みが書いてあるんだ。
*   **注意！:**
    *   このファイルをいじるのは、プログラムのことが少し分かる人向けだよ。間違って書き換えると、動かなくなっちゃうかもしれないから、気をつけてね！
    *   もし項目を増やしたり消したりしたら、他のファイルも直さないといけないことがあるから、分かる人に相談するのが良いかもしれないね。

## 🤔 困ったときは？ (Q&A)

*   **Q1. 解析ボタンを押しても、何も動かない！エラーも出ない！**
    *   A1. まず、裏側のサーバーがちゃんと動いてるか確認してね！ 起動したときの黒い画面を見て、エラーが出てないかな？ 「Running on...」って出てる？ それと、インターネットに繋がってるかも確認してね。もう一度試してみてね。
*   **Q2. 解析は終わったけど、結果が変だよ…AIってバカなの？**
    *   A2. AIも人間と同じで、たまに間違えたり、変なことを言ったりするんだ。ごめんね！ あと、入力した情報が分かりにくかったり、評価のルールが曖昧だったりすると、うまく評価できないこともあるんだ。ルールを見直してみるのも良いかもしれないね。
*   **Q3. さっきまで結果が表示されてたのに、ページを更新したら消えちゃった！**
    *   A3. そうなんだ、このアプリは結果を保存してないから、ブラウザを閉じたり、ページを更新したら消えちゃうんだ。大事な結果は、こまめに「エクスポート」ボタンを押して、ファイルで保存しておいてね！
*   **Q4. CSVファイルが文字グチャグチャで読めない！**
    *   A4. あー、文字化けだね。Excelで開くとき、普通にダブルクリックするんじゃなくて、「データ」タブ → 「テキストまたはCSVから」を選んでみてくれる？ 途中で文字コードを選ぶ画面が出るから、「65001: Unicode (UTF-8)」ってやつを選んでから「読み込み」だよ！ これで大体読めるようになるはずだよ。
